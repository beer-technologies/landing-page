## Getting Started

First, install dependencies and start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open your browser and navigate to:

```text
http://localhost:3000
```

You can begin editing the landing page by modifying:

```text
pages/index.tsx
```

The page will auto-update as you save changes.

---

## API Routes

API routes are located in the `pages/api` directory and are automatically mapped to `/api/*`.

Example:

```text
http://localhost:3000/api/base
```

You can edit this endpoint in:

```text
pages/api/base.ts
```
